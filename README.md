# BookMySpace

BookMySpace 是一個會議室預約系統，用於顯示並管理會議室的預約狀態。用戶可以根據會議室、時間、日期查看預約情況，並進行預約管理。

## 專案介紹

BookMySpace 是一個針對公司內部會議室管理的工具。它能夠：
- 以 **會議室分類**，通過 **Tab** 查看預約狀況。
- 顯示 **未來14天** 的預約狀況，並提供具體的日期、時間段、會議室狀態、部門和預訂者姓名。

## 功能

- 會議室分類查看
- 按日期和時間段查看會議室狀態
- 預約信息展示，包括狀態、部門、預訂者
- 移動端響應式設計，適合各類設備
